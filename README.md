# amazon-product-performance-analysis-sql
SQL-based analysis of Amazon product listing data to evaluate product performance, customer satisfaction, manufacturer performance, pricing patterns, and inventory availability using MySQL.
# Amazon Product Performance Analysis Using SQL
This project explores Amazon product listing data using SQL in MySQL Workbench. The goal was to understand product performance by analyzing customer ratings, manufacturer performance, pricing, and stock availability.
The dataset was cleaned in Excel before being imported into MySQL. Using SQL queries such as `SELECT`, `WHERE`, `GROUP BY`, `ORDER BY`, `COUNT()`, `AVG()`, and `CASE`, I analyzed the data and generated business insights.
## Files
-`Amazon_Project.sql` – SQL queries used for the analysis
- `Amazon_Product_Performance_Analysis_Report.pdf` – Complete project report
- `dataset/` – Dataset used for the project
## Key Highlights
- Analyzed 122 Amazon products
- Identified top-performing manufacturers and products
- Compared ratings across different price categories
- Analyzed stock availability and customer satisfaction
- Generated business insights using SQL
For detailed methodology, queries, screenshots, and findings, please refer to the project report.
